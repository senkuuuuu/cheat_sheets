## 🔧 Installation

To install Python3, use the following commands based on your operating system:

### **Windows**
```bash
# Download the installer
https://www.python.org/downloads/

# Verify installation
python --version

```
### **Linux**
```bash
# Install Python3
sudo apt update
sudo apt install python3

# Verify installation
python3 --version

```
